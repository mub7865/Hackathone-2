# Phase I: Python Console App

## Overview
In-memory Python todo application with basic CRUD operations.

## Features
- Add tasks
- Delete tasks
- Update tasks
- View task list
- Mark tasks as complete

## Technology Stack
- Python 3.13+
- UV package manager
- Pytest for testing

## Running
```bash
cd phase1-console
uv sync
python src/cli/main.py
```

## Testing
```bash
pytest tests/
```
